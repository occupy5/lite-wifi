# lite-wifi
## 场景
春节期间，家里会来很多亲戚。无论大人小孩，来家里的第一件事就是要wifi密码。以前有两种方式实现这个需求，一是我来帮他输入。二是，我告诉他密码，他来输入（容易输错）。
更优雅的方式：放一张二维码，让客人扫描。进入页面，点击复制按钮。粘贴密码，连接wifi。
